Front-End

O desenvolvimento front-end da web é o desenvolvimento da interface gráfica do usuário de um site por meio do uso de HTML , CSS e JavaScript para que os usuários possam visualizar e interagir com esse site.

Back-End

Back-end é a parte interna de uma aplicação, que é responsável por gerenciar a funcionalidade geral do site ou aplicativo. É o que acontece nos bastidores, sem que o usuário veja.

UX - UI

UX e UI são áreas do design que se referem à experiência do usuário (UX) e à interface do usuário (UI). O objetivo é criar produtos que proporcionem a melhor experiência possível para o usuário.

Data Science

Data science, ou ciência de dados, é a área de estudo que analisa dados para obter informações úteis para os negócios. É uma abordagem multidisciplinar que combina matemática, estatística, inteligência artificial e engenharia da computação.
